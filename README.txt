Evaluate the performance of your Staffs/Employees easily.

== Description ==

Supercharge your HR by measuring their performance from your WordPress Dashboard by just a few clicks.

* Features
-> Calculate the performance based on 8 Advanced criterias!
* Job Knowledge
* Work Quality
* Attendance/Punctuality
* Communication/Listening
* Learning Capability
* Responsibility
* Confidence Level
* Dedication Level

== Installation ==

1. Upload `hr-performance.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
